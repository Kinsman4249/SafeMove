# Contributing to SafeMove

SafeMove is a small, safety‑focused PowerShell project. Contributions are welcome, but safety and predictability come first.

## Guidelines

- Do not introduce behaviours that could overwrite or delete destination data
- Maintain PowerShell 5.1 compatibility
- Test with `-WhatIf`

## Submitting Changes

Fork the repo, create a feature branch, test thoroughly, and open a PR against main.
